
# CRITICAL DATABASE FIX (DUPLICATE KEY ERROR)

If you see "E11000 duplicate key error... email_1 dup key":

**YOU MUST CHANGE THE DATABASE NAME IN VERCEL.**

1. Go to Vercel Dashboard -> Your Project -> Settings -> Environment Variables.
2. Edit `MONGO_URI`.
3. Look for the part that says `...mongodb.net/test?...` or `...mongodb.net/myFirstDatabase?...`.
4. CHANGE the word after the slash to something unique.
   Example: `...mongodb.net/july6app?...`
5. Save and Redeploy (or just restart the app).

This forces MongoDB to create a fresh database without the old "email" requirement that is breaking your user registration.
